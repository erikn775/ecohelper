# Eco Helper React App

This app was created using a React frontend and a Rails Api backend.

The purpose is to help users learn tips about how to make their life more energy efficent.
This is not an extensive list but more of a conversation starter on home to lower your impact.

## yarn start or npm start

Open http://localhost:3000 to view it in the browser.

Also must run the back end to view this app because it is not deployed yet
Backend Github: https://github.com/erikn775/react_app_backend
